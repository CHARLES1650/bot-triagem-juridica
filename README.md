Bot de Triagem Jurídica

## Descrição
Este projeto consiste em um chatbot desenvolvido para realizar triagem inicial de casos jurídicos.

## Objetivo
Simular o atendimento inicial de um cliente, identificando a área do Direito, sugerindo medidas jurídicas e orientando quanto aos documentos necessários.

## Funcionalidades
- Identificação da área do Direito
- Sugestão de medidas jurídicas
- Indicação de documentos necessários
- Avaliação de urgência
- Limite de 3 interações com resumo final

## Tecnologias utilizadas
- Python
- OpenAI API
- Google Colab

## Como executar
1. Inserir a API Key
2. Executar o notebook no Google Colab

## Exemplo
Entrada:
"Fui negativado indevidamente"

Saída:
Área do Direito: Consumidor  
Medidas possíveis: Ação indenizatória  
Documentos necessários: Comprovantes  
Urgência: Alta
